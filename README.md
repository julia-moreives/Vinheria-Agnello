# 🍷 Vinheria Agnello

## 📌 Descrição do Projeto

O projeto **Vinheria Agnello** foi desenvolvido com o objetivo de criar um site institucional para uma vinheria fictícia localizada em São Paulo.

A proposta simula um cenário real, onde a empresa deseja divulgar seus produtos, promoções e informações institucionais, além de oferecer um canal de contato com os clientes.

O sistema foi construído utilizando HTML e CSS, aplicando conceitos fundamentais de desenvolvimento web como estruturação semântica, navegação entre páginas e estilização visual.

---

## 🧱 Estrutura do Projeto

O site é composto por 5 páginas principais:

* **index.html (Página Inicial)**
  Apresenta a vinheria, com destaque para o diferencial da empresa e um vídeo institucional.

* **produtos.html**
  Exibe os tipos de vinhos disponíveis, organizados em categorias, além de uma tabela com sugestões de harmonização.

* **promocoes.html**
  Mostra ofertas e combos promocionais disponíveis para os clientes.

* **sobre.html**
  Contém informações institucionais da vinheria, como história, valores e equipe.

* **contato.html**
  Disponibiliza um formulário para que o usuário possa entrar em contato com a empresa.

---

## 🎨 Estilização (CSS)

O projeto utiliza dois arquivos CSS com funções diferentes:

* **style.css (CSS principal)**
  Responsável pela estrutura visual do site, incluindo cores, tipografia, espaçamentos, layout e organização dos elementos.

* **efeito.css (CSS de interatividade)**
  Responsável pelos efeitos visuais e interações com o usuário, como:

  * Pseudo-classes (`:hover`, `:active`, `:focus`)
  * Pseudo-elementos (`::after`)
  * Animações com `@keyframes`
  * Transições e pequenas transformações em elementos interativos

Essa separação foi feita para organizar melhor o código e facilitar a manutenção.
